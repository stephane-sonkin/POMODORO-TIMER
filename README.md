## ✨ Features

- ⏱️ Work / Break mode switching
- 🔁 Automatic session handling
- ▶️ Start / Pause / Reset controls
- 📊 Session tracking
- ⚡ Real-time countdown
- 🔔 Sound notification when the timer ends

---

## 🛠️ Technologies

- HTML5 — Structure
- CSS3 — Styling & animations
- JavaScript (Vanilla) — Timer logic
